# Para compilar o compilador

make

# Para compilar um programa da linguagem da disciplina

`./etapa7 <caminho do arquivo>`

Um assembly será gerado com nome `<caminho do arquivo>.s`, um executável também
é gerado. O arquivo assembly é gerado independentemente de falha na geraçãodo
executável. O arquivo final chamará-se `a.out`.

# Para ligar todas as otimizações

`./etapa7 -O <caminho do arquivo>`

# Para printar TACs na tela

`./etapa7 -t <caminho do arquivo>`

# Para printar TACs na tela com otimizações

`./etapa7 -t <caminho do arquivo> -O`

# Para obter mais ajuda de como usar o compilador

`./etapa7 --help`
